# laravel-BRIT

The skeleton application for the Laravel framework with modifications. BRIT stands for Breeze, React, Inertia, Tailwind.

## Installation

```shell
composer create-project msamgan/brit my-app
```

## Usage

```shell
cd my-app
npm install
npm run dev
php artisan serve
```
